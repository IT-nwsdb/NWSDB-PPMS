Rehabilitation Budget – Data Entry System (v4 Standalone)

This version is OFFLINE and has NO Bootstrap/CDN, so it won't show tracking-prevention warnings.

Run:
1) Unzip
2) Open index.html in Edge/Chrome

Features:
- Add Activity + Quick Add
- Add Region / Add Category
- Filters + Search
- Summary totals + progress
- Click a row to preview (right side drawer)
- Theme toggle + compact table + print
- No Export/Import

Storage:
- Saves locally (tries localStorage; if blocked, uses in-memory for the session).

# Dead Simple Project Entry (Excel style)

## What this version does
- ONLY 4 buttons: **+ Project**, **+ Row**, **Save**, **Reset**
- **Blue row = project name**
- Fill activities under the project like Excel
- Click the blue row to select the project (then + Row adds under it)
- Data auto-saves in browser (and Save button shows confirmation)

## Run
1) Unzip
2) Open `index.html` in Chrome/Edge

Tip: VS Code → Live Server extension.


## Compact layout changes
- Columns are fixed width and text wraps.
